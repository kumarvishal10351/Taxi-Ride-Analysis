# 🚖 Taxi Ride Analytics Dashboard (Power BI)

An interactive and visually rich **Power BI dashboard** designed to analyze taxi ride data, providing insights into revenue, bookings, cancellations, ratings, and vehicle performance.

---

## 📊 Overview

This project presents a comprehensive analysis of taxi ride operations using **Power BI**. It enables users to explore key metrics and uncover trends that help in decision-making for transportation services.

---

## 🚀 Features

* 📈 **Revenue Analysis** – Track earnings over time
* 🚕 **Vehicle Type Insights** – Compare performance across vehicle categories
* ❌ **Cancellation Analysis** – Understand reasons and trends in ride cancellations
* ⭐ **Customer Ratings** – Evaluate service quality
* 📅 **Time-Based Trends** – Analyze peak hours and ride patterns
* 📊 **Interactive Visualizations** – Dynamic filtering and drill-down capabilities

---

## 🧩 Dashboard Pages

* **Homepage** – Overview and navigation
* **Overall** – Key performance metrics
* **Vehicle Type** – Vehicle-wise breakdown
* **Revenue** – Revenue trends and insights
* **Cancellation** – Cancellation patterns and reasons
* **Ratings** – Customer feedback analysis
* **Summary** – Consolidated insights

## 📌 Data Insights

The dashboard provides insights such as:

* Total bookings and cancellations
* Revenue distribution across time and vehicle types
* Peak ride hours (`Is_Peak_Hour`)
* Payment method trends
* Ride distance and pricing patterns
* Service quality indicators
